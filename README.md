Este é um pipeline de análise em Python projetado para automatizar o processamento de dados experimentais de biossensores ópticos (como LPGs - Long-Period Gratings).

O projeto transforma dados brutos — uma planilha Excel com metadados e pastas de arquivos de espectro (.txt) — em um conjunto completo de análises, incluindo curvas de calibração, análise temporal e cálculo do Limite de Detecção (LOD).
